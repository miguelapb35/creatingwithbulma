Columns

Flexbox is a one-dimensional grid system, providing you with either rows or columns. In Bulma, you develop websites with columns in mind and wrap your columns inside a row or wrapper.

To wrap the columns use a

div class="columns":

<div class="columns"> <div class="column"></div> <div class="column"></div> <div class="column"></div> </div>

Components

Bulma ships with components, which are pre-styled chuncks of code that serve a certain purpose. With components, you have to follow a specific HTML structure. Reference Bulma's documentation for more information and examples of components. Here is an example of a card component:

<div class="card"> <header class="card-header"> header content </header> <div class="card-content"> <div class="card-image"> card-image </div> </div> <footer class="card-footer"> footer content </footer> </div>
